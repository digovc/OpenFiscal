#OpenFiscal
###Biblioteca para gerar, autorizar e transmitir arquivos fiscais brasileiros.

##Ajuda:

###Criar arquivo XML da Nota Fiscal eletrônica

Síntese:
`openfiscal [-x|--xml "<diretório>"]`

Descrição: Comando para criação do arquivo XML que será posteriormente assinado e transmitido para o Governo.
Este comando tem um argumento que é o diretório onde está localizado o objeto "NotaFiscalEletronica" em forma de XML (serializado) devidamente preenchido.
